# 明鉴草书黑体

*（感谢[NaitLee](https://unseen-site.fun/about/#reach-out)基于本站制作了网页版，您可以[点击此处](https://草书.unseen-site.fun/font/alpha-download.html)浏览。）*

这是一款笔画清晰明确的开源草书黑体字库。字库为下列用途精心设计：
* **书法参考**
	* 字形**清晰明了**，以黑体作为表现形式，呈现今草字形。具有传统字帖和书法字典无法相比的**清晰性**、**系统性**和**规范性**。
	* 字形**严谨考究**，字字用形参考两千年各代书法名家，兼逐字严格分析构型结构，呈现最为地道规范草书字形，规避了以往字帖和参考资料中充斥的异写形和讹形。一些有多种写法的字，还补充有额外的可选字形。
* **草书初学**
	* 字形**识别度高**，适合初学熟悉草书的爱好者在日常使用中快速熟悉草书字形。
* **日常使用**
	* 字形**简洁易读**，具备信息时代黑体字的美观性，非常适合在**电子设备**、**招牌海报**中取代一黑体字大量使用。
* **艺术设计**
	* 字形**现代简约**，适合取代楷书黑体字出现在各种应用场景。字库完全开源，可以免费自用商用。

## 完成进度

目前字体尚在完成中，已经覆盖了3000余汉字。计划覆盖现代汉语常用字表（3500字）、国标2312（6763字）、国标12345（6866字）、常用國字標準字體表（4808字）、常用字字形表（4762字）、常用漢字表（日本，2136字）、漢文教育用基礎漢字（韓國，1800字），共约10000字符。

现在字体字形已经进入批量生产阶段，持续更新。

如您急需已完成的字库，可以在[方正字库官网](https://www.foundertype.com)获取「方正巴龙草书黑体」。这是一款之前由我主导设计的字库。它不是开源的，请与版权方沟通具体使用条款。

## 制作流程

本开源库除了发布明鉴草书黑体，也意在分享字体（草书）制作的流程。您可以在[这个文档](WorkFlow.md)找到字体制作流程的描述。

因为本字体的诞生较为坎坷，所以制作流程并不十分完整。作者计划以后继续制作其它草书字体，届时会分享更加完整优化的制作流程。
